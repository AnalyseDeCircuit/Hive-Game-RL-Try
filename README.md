# Hive_Python

这是一个基于 Python 的 Hive 游戏实现项目。

## 项目结构

- `main.py`：程序入口，启动游戏。
- `game.py`：游戏主逻辑。
- `board.py`：棋盘相关逻辑。
- `piece.py`：棋子相关逻辑。
- `player.py`：玩家相关逻辑。
- `utils.py`：工具函数。

## 运行方法

1. 确保已安装 Python 3.7 及以上版本。
2. 在命令行中进入项目目录。
3. 运行：

```powershell
python main.py
```
